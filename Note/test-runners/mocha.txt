# mocha: jest,ava
    - test runner framework
    - describe and it is a function provided by mocha.

# assert: node built-in assertion module
# chai : assertion library
# sinon: assertion plus test doubles library

------------------------------------------------------------------------------------------------------

- mocha is a light-weight Node.js test framework
- chai is a BDD / TDD assertion library for node and the browser that can be delightfully paired with any javascript testing framework.
- BDD provides an expressive and readable style of language via Should & Expect, whereas TDD provides a more Classical approach via Assert.
- Both Mocha and Chai run in NodeJs and the browser and allow asynchronous testing.

# Terms:
    - asynchronous testing
